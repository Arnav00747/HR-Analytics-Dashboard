# HR Analytics Dashboard 📊

## 📌 Overview

This project is a Power BI dashboard that analyzes employee attrition, salary trends, and workforce insights.

## 🔧 Tools Used

* Power BI
* Excel / CSV

## 📊 Key Insights

* Total Employees: 1470
* Attrition Count: 237
* Attrition Rate: 16.1%
